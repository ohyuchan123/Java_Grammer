## enum
```
final로 String과 같은 문자열이나 숫자들을 나타내는 기본 자료형의 값을 고정할 수 있다
이렇게 고정된 값을 좀 유식하고 간단한 말로 하면 "상수"라고 하고 영어로는 constant라고
한다. 그런데, 어떤 클래스가 상수만으로 만들어져 있을 경우에는 반드시 class로 선언할 
필요는 없다. 이럴 때 class라고 선언하는 부분에 enum이라고 선언하면 "이 객체는 상수의
집합이다"라는 것을 명시적으로 나타낸다. enum은 enumeration이라는 "셈,계산,열거,목록,
알람표"라는 영어 단어의 앞부분만 따서 만들어진 예약어이다.

먼저 enum 클래스는 어떻게 보면 타입이지만, 클래스의 일종이다. 그래서 enum 클래라고
부르면 된다. 한글로는 "열거형" 클래스라고 불러도 무방하다.
```
